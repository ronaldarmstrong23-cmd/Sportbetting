# Game Analysis Research Toolkit

This repository stores a neutral research workflow for analyzing games, player availability, team trends, projected roles, and performance outcomes.

## Project Purpose

The goal is to keep research organized in one place so each game review follows the same process. The workflow focuses on current data, injury status, expected roles, historical trends, and post-game review.

## Core Workflow

1. Pull the current game slate.
2. Confirm player availability and rest risk.
3. Identify starters, bench roles, and expected minutes.
4. Review recent team and player trends.
5. Grade each research angle using a confidence scale.
6. Separate notes into Core, Watch, High-Variance, and Avoid groups.
7. Save final notes in the daily research log.
8. Review results after games are completed.

## Confidence Scale

| Grade | Score | Use Case | Description |
|---|---:|---|---|
| Core | 8-10 | Strongest research angles | Stable role, strong matchup, low uncertainty |
| Watch | 6-7 | Usable but needs monitoring | Good angle with some role, injury, or matchup risk |
| High-Variance | 4-5 | Small sample or volatile angle | Higher uncertainty, bench role, or unclear minutes |
| Avoid | 1-3 | Do not use | Poor data support, uncertain role, bad matchup, or major injury risk |

## Recommended File Structure

- `research-rules.md` — Core research rules and grading filters
- `daily-research-log.csv` — Daily tracking sheet
- `availability-report-template.csv` — Player availability and role tracking
- `player-metric-grading-template.csv` — Player metric review sheet
- `risk-management-guide.md` — Research discipline and uncertainty control
- `data-sources.md` — Preferred research sources
- `requirements.txt` — Optional Python packages for future scripts

## Review Checklist

Before finalizing any research note, confirm:

- Current player availability
- Expected starters
- Expected minutes or role
- Recent three-to-five-game trend
- Matchup fit
- Team motivation
- Schedule fatigue or rest concern
- Any late news that changes the angle

## Reminder

This project is for research, tracking, and educational analysis. No projection is guaranteed. Always update notes when new availability or lineup information is released.
